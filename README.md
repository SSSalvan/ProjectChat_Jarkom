# Socket-Based Chat Application

## Overview
A real-time, multi-client chat application developed to demonstrate fundamental computer networking concepts. The system relies on low-level TCP/IP sockets to establish persistent connections between a centralized server and multiple distributed clients.

## Technical Architecture
- **Protocol:** TCP/IP Sockets
- **Language:** Python
- **Architecture:** Client-Server Model with Multi-threading

## Features
- Concurrent handling of multiple client connections using threads.
- Real-time message broadcasting and direct messaging logic.
- Graceful connection termination and error handling for network drops.
